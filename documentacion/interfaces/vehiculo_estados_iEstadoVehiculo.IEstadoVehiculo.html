    Interface IEstadoVehiculo

    Interfaz que define el comportamiento que debe tener un vehículo en función de su estado actual (Patrón State).

    Cada clase concreta que implemente esta interfaz define la lógica de negocio y las posibles transiciones de estado para las acciones de alquiler, devolución y mantenimiento.

    interface IEstadoVehiculo {
        alquilar(
            vehiculo: Vehiculo,
            cliente: default,
            fechaInicio: Date,
            fechaFin: Date,
        ): void;
        devolver(vehiculo: Vehiculo): void;
        finalizarMantenimiento(

            costo: number,
            fechaFin: Date,
        ): void;
        iniciarMantenimiento(vehiculo: Vehiculo, fechaInicio: Date): void;
    }
